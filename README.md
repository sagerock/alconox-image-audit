# Alconox TechNotes — image license audit

Visual triage of every image used in a published TechNote (technotes.alconox.com),
grouped by how confident we are that it was paid for / is ours. Open **index.html**
(or the GitHub Pages link) and start at the top: the 🔴 SUSPECT section is the worklist.
Each image has TinEye / Google Lens reverse-search buttons and a link to its post.
